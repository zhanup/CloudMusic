# CloudMusic

仿网易云音乐的微信小程序

接口使用的是[netmusic-node](https://github.com/sqaiyan/netmusic-node)

项目比较简单，只有五个页面，包括推荐音乐、热歌榜、搜索、歌单列表、音乐播放。
